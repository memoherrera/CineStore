## Usage

1. **Clone the Repository**:

    ```bash
    git clone git@github.com:memoherrera/CineStore.git
    cd BaseSwiftUI
    ```
2. **Open Xcode** 

    Open Xcode project:

    ```bash
    open CineStoreApp.xcodeproj
    ```
    
3. **Build and Run**:

    Build and run the project in Xcode.

## Project Folder Organization
